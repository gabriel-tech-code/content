## Project: TranslaGenix — Multilingual Translation Platform
A team‑built application designed to translate text, images, speech, and files into multiple languages using external APIs.

### **Project Summary**
TranslaGenix enables users to translate the world around them. The application accepts input from text, images, speech, or uploaded files and returns translations in the user’s preferred language. The goal was to create an accessible tool that breaks language barriers and supports global communication.

### **Key Responsibilities**
- Developed backend services in **C#** and **ASP.NET Core**
- Integrated multiple external APIs for translation, speech‑to‑text, and image processing
- Built responsive UI components using **Angular**, **TypeScript**, and **Bootstrap**
- Implemented secure authentication using **Okta**
- Improved code quality and maintainability using **SonarCloud**
- Deployed services using **Docker** and Azure cloud resources

### **Tech Stack**
**Backend:** C#, .NET 6, ASP.NET Core Web API, EF Core  
**Frontend:** Angular, TypeScript, HTML, CSS, Bootstrap  
**APIs Used:**  
- Microsoft Translation API  
- Microsoft Computer Vision API  
- Speech‑to‑Text API  
- Okta Authentication API  

**Infrastructure:** Docker, Azure SQL, Azure App Service  
**CI/CD:** GitHub Actions  
**Code Quality:** SonarCloud
